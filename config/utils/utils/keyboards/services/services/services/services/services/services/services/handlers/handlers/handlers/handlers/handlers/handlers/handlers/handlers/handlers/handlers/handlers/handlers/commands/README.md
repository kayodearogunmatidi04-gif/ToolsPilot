# ToolPilot Bot

A multi‑tool Telegram bot built with Node.js and Telegraf.

## Features

- 📝 Text Tools (Uppercase, Lowercase, Title Case, Sentence Case, Remove Spaces, Reverse)
- 🔐 Password Generator (customizable length, symbols, numbers, uppercase)
- 📷 QR Generator (text → QR image)
- 🔢 Random Generator (Number, Password, PIN, Color Hex)
- 🆔 UUID v4 Generator
- 🔤 Base64 Encode/Decode
- 🔒 Hash Generator (MD5, SHA1, SHA256)
- 📊 Word Counter (words, characters, paragraphs, reading time)

## Commands

/start – Main menu
/help – Help
/about – About
/tools – Tools menu
/password – Password generator
/qr – QR generator
/text – Text tools
/random – Random generator
/uuid – UUID generator
/base64 – Base64 tools
/hash – Hash generator
/counter – Word counter
